README.TXT
==========
This is Team 17's submission for CS6310's Project2: Heated Earth Simulation.

To build the code, run the following commands in bash:

	./build.sh

To run the code, run the command expected in the instructions from bin dir, ie,
   
	cd bin
	java EarthSim.Demo [-s] [-p] [-r|-t] [-b #]

Alternatively use the provided run script:

	./run.sh
